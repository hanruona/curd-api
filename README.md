# curd-api
重点学生成绩录入

### 全局安装
npm install dva-cli -g

### 创建项目
dva new dva-quickstart
cd dva-quickstart

### 启动
npm start

### 引入创建的module
app.model(require('./models/products').default);

### 连接数据库
import { connect } from 'dva';
